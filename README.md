Helios Ultra Cleaner
[TR] Helios Ultra Cleaner, Windows işletim sistemini yavaşlatan gereksiz dosyaları, önbellek verilerini ve eski güncelleme artıklarını güvenli bir şekilde temizlemek için geliştirilmiş, gelişmiş ve taşınabilir bir sistem bakım aracıdır.

[EN] Helios Ultra Cleaner is an advanced and portable system maintenance tool developed to safely clean up junk files, cache data, and obsolete update remnants that slow down the Windows operating system.

🛠 Temizlik Modları / Cleaning Modes
MOD 1: HIZLI (FAST)

[TR] Günlük kullanımda oluşan geçici dosyaları (Temp) siler.

[EN] Cleans temporary files (Temp) generated during daily use.

MOD 2: DERİN (DEEP)

[TR] DNS önbelleğini temizler, Geri Dönüşüm Kutusu'nu boşaltır ve Windows Update artıklarını siler.

[EN] Flushes DNS cache, empties the Recycle Bin, and removes Windows Update leftovers.

MOD 3: ULTRA TEMİZLİK (ULTRA CLEAN)

[TR] Sistem imajı bileşenlerini ve yıllardır biriken eski güncelleme yedeklerini temizler. Ciddi disk alanı açar.

[EN] Cleans system image components and accumulated old update backups. Frees up significant disk space.

🚀 Nasıl Kullanılır / How to Use
İndirin: ZIP dosyasını indirin ve istediğiniz bir klasöre çıkartın. / Download the ZIP file and extract it to a folder.

Çalıştırın: HeliosUltraCleaner.cmd dosyasına sağ tıklayıp "Yönetici olarak çalıştır" deyin. / Right-click HeliosUltraCleaner.cmd and select "Run as administrator".

Seçin: Arayüzdeki modlardan ihtiyacınız olan temizlik seviyesini seçin. / Select the cleaning level you need from the interface buttons.

İzleyin: İşlem bittiğinde "İşlem Tamamlandı" mesajını bekleyin. / Wait for the "Islem Tamamlandi!" message to appear.

🛡 Güvenlik ve Şeffaflık / Security and Transparency
[TR] Yazılım, Windows'un resmi sistem komutlarını (dism, cleanmgr, del) kullanır. Kişisel dosyalarınıza veya sistemin çalışması için kritik olan dosyalara zarar vermez.
[EN] The software uses official Windows system commands (dism, cleanmgr, del). It does not harm your personal files or critical system files.
